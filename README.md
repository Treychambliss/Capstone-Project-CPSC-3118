# Capstone Project (Fahrenheit to Celsius Converter)

## Description
A simple desktop application that converts temperatures from Fahrenheit to Celsius, demonstrating MVC architecture.

## MVC Overview
- Model: Handles the temperature conversion math and logic.
- View: GUI built in Qt Designer containing the inputs, labels, and buttons.
- Controller: Connects the GUI buttons to the model's conversion updates.

## How to Run
python main.py
